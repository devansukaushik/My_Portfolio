Project Overview

In my project, I have used React to build the client-side interface of the chat application & Node.js and Express to build the server-side of the application, which will handle user authentication and message exchange. I have used Socket.io to enable real-time communication between the server and client.
Finally, I have used MongoDB to store and retrieve messages exchanged between users. This will enable users to view their chat history even after logging out of the application.

React is a JavaScript library used for building user interfaces. It enables me to create reusable UI components and render them efficiently.
Node.js is an open-source, cross-platform JavaScript runtime environment that enables me to build server-side applications using JavaScript. Socket.io is a JavaScript library that provides real-time, bidirectional communication between web clients and servers. It uses WebSockets to enable real-time data exchange. MongoDB is a NoSQL document-oriented database that stores data in JSON-like documents. It provides scalability, high availability, and automatic data sharding. Express is a minimal and flexible Node.js web application framework that provides a set of features for my web and mobile applications.
